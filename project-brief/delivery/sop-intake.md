# SOP — Client Intake (45 minutes)

1. Confirm scope: FTD includes 1 workflow, Teams onboarding template, SharePoint queue, runbook, rollback.
2. Collect: HRIS API keys or export, license SKUs, org units, manager list, security constraints.
3. Access: grant least‑privileged roles to sandbox tenant; set up Managed Identity and Key Vault.
4. Baseline: screen capture current onboarding; capture steps and timings.
5. Choose edge cases: interns, contractors, license scarcity, off‑cycle hires.
6. Agree on KPI baseline and measurement window (14–30 days). Document in kickoff note.
