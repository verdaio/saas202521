# SOP — Deploy (Day 2 Afternoon)

1. Export tested flow, import to production environment.
2. Rebind connections to production resources (SharePoint site, Key Vault, Function URL).
3. Run 1 live hire under supervision.
4. Record a 5‑minute how‑to video for HR and managers.
5. Hand over runbook and rollback plan. Confirm success criteria and schedule the 2‑week review.
